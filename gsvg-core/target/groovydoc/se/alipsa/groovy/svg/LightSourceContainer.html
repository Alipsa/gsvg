[Groovy] Trait LightSourceContainer<T extends FilterElement<T>>

    • se.alipsa.groovy.svg.LightSourceContainer 
    • @groovy.transform.Trait
@groovy.transform.CompileStatic
@groovy.transform.SelfType(value: FilterElement)
trait LightSourceContainer<T extends FilterElement<T>>
extends java.lang.Object

      Trait for filter primitives that can contain light source elements.

      This trait uses the @SelfType annotation to declare that any class implementing this trait must extend FilterElement. This allows the trait to call methods from FilterElement (such as addAttribute()) while maintaining type safety under static compilation. Without @SelfType, the static compiler would not be able to verify these method calls.
