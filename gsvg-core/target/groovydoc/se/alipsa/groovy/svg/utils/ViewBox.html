[Groovy] Class ViewBox

    • se.alipsa.groovy.svg.utils.ViewBox 
    • @groovy.transform.CompileStatic
@groovy.transform.Immutable
class ViewBox
extends java.lang.Object

      Represents an SVG viewBox attribute with helper methods for manipulation and conversion.

      The viewBox attribute defines the coordinate system and aspect ratio for an SVG element. It consists of four values: minX, minY, width, and height.

      Example usage:

       // Create a viewBox
 def vb = ViewBox.of(0, 0, 100, 100)

 // Parse from string
 def parsed = ViewBox.parse("0 0 200 200")

 // Transform it
 def scaled = vb.scale(2.0)        // 0 0 200 200
 def translated = vb.translate(10, 10)  // 10 10 100 100

 // Use with SVG
 svg.viewBox(vb.toString())
 // or with convenience method:
 svg.viewBox(vb)

        • ViewBox fitToContain(ViewBox other, boolean preserveAspectRatio)

          Scale and center another viewBox to fit inside this viewBox.

          When preserveAspectRatio is true (default), 'other' is uniformly scaled to fit inside this viewBox while maintaining its aspect ratio, then centered.

          When preserveAspectRatio is false, 'other' is scaled non-uniformly to exactly fill this viewBox. The returned ViewBox has this viewBox's position and dimensions, representing where 'other' would appear after being stretched/squashed to fill.

          Parameters:
          other - the viewBox to scale and fit inside this one
          preserveAspectRatio - whether to preserve aspect ratio (if false, scales to fill)
          Returns:
          a new ViewBox representing the fitted result
