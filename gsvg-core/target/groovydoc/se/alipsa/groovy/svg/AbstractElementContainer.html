Package: se.alipsa.groovy.svg

[Groovy] Class AbstractElementContainer<T extends AbstractElementContainer<T>>

      • se.alipsa.groovy.svg.AbstractElementContainer 
      • @groovy.transform.CompileStatic
abstract class AbstractElementContainer<T extends AbstractElementContainer<T>>
extends SvgElement<T>

        Base class for SVG elements that can contain child elements and provide factory helpers.
