# Security Policy

## Supported Versions

Security fixes are provided for the latest minor release line.

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

Please report security issues privately.

Preferred: open a GitHub Security Advisory for this repository.
If that is not available, open an issue with minimal details and request a private channel.

We aim to acknowledge reports within 72 hours and provide updates as fixes progress.
